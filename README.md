# Html Composable for Android Jetpack Compose

&copy; 2022 SiLeader.

## Features

+ Display HTML document

### Supporting HTML tags

+ block elements
    + `body`
    + `h1`
    + `h2`
    + `h3`
    + `h4`
    + `ol`
    + `ul`
    + `p`
    + `table` (Using `AndroidView`)
+ inline elements
    + `span`
    + `a`
    + `em`
    + `strong`

### Supporting inline style

+ `#000000` styled `color` and `background-color`
+ `margin`, `padding`
+ `font-size`
+ `text-decoration`
    + `line-through`
    + `underline`
+ `text-align`
